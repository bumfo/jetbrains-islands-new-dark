# Theme Project

IntelliJ IDEA color scheme (.icls) files. Both are Darcula-based dark themes.

## Files

- `Islands_Dark.icls` — JetBrains Islands Dark theme (warm orange keywords, blue functions, extensive language-specific tokens)
- `Islands_New_Dark.icls` — New Dark theme (cool blue keywords, gold functions, purple parameters, more minimal token set)

## Key Palette Differences

| Token         | New Dark        | Islands Dark    |
|---------------|-----------------|-----------------|
| Keyword       | blue `749ded`   | orange `cf8e6d` |
| Function      | gold `debc7e`   | blue `56a8f5`   |
| String        | green `7ec482`  | green `6aab73`  |
| Number        | pink `d981de`   | teal `2aacb8`   |
| Instance field| cyan `72cfd6`   | purple `c77dbb` |
| Parameter     | purple `b189f5` | default text    |
| Background    | `1e1f22`        | `191a1c`        |
